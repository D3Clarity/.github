# D3Clarity — org community health files

This repository holds D3Clarity's **org-wide community health files**. GitHub uses
files here as **defaults for every D3Clarity repository** that doesn't define its
own.

Contents:
- **`.github/ISSUE_TEMPLATE/`** — standardized issue Forms (`Bug`, `Feature`,
  `Task`) that set the native Issue Type and enforce our issue body standard,
  plus `config.yml` (blank issues disabled; security + standard links).
- **`ISSUE_TRACKING.md`** — the org **issue tracking & labeling standard**
  (native types/fields vs labels, lifecycle, conventions).
- **`SECURITY.md`** — how to report vulnerabilities (privately, never as a public issue).

> This repo is intentionally **public** — that is required for GitHub to apply
> these templates org-wide. It contains process/templates only: **no code, no
> configuration, no secrets.**
